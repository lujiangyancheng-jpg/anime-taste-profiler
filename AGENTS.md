# Repository Instructions

- Treat `anime-taste-profiler/` as the main app source for this project.
- For future Codex changes in this repository, work in the GitHub-backed checkout and keep the remote repository in sync.
- Unless the user explicitly asks for local-only work, finish implementation tasks by checking the changed files, running the most relevant validation available, committing the intended changes, and pushing to the current remote branch.
- Do not commit generated caches, credentials, virtual environments, downloaded datasets, or unrelated local files.
- If the working tree contains unrelated user changes, do not stage them silently; ask which files belong in the GitHub sync.
